## Users

OK  "/api/user/menu/" [GET](admin)
    JSON list of user api endpoints

OK  "/api/user/" [GET, POST](admin)
    JSON list of users
    create new user (signup)(any)

OK  "/api/user/<slug>/" [GET, PUT, PATCH, DELETE](admin)
    JSON user detail, delete or update a user

OK  "/api/user/<slug>/public/" [GET](any)
    JSON user public datas

OK  "/api/user/current/" [GET, PUT, PATCH](auth)
    JSON current (authenticated) detail and update

OK  "/api/user/token/" [POST](any)
    Get jwt auth and refresh token from credentials

OK  "/api/user/token/refresh/" [POST](auth)
    Get refreshed auth jwt token from refresh token

    "/api/user/recoverlink/" [POST](any)
    Send a mail with UUID from username or email

    "/api/user/recover/" [POST](any)
    Update user password with recovering 



## Pins

OK  "/api/pin/menu/" [GET](admin)
    JSON list of pin api endpoints

OK  "/api/pin/" [GET, POST] (any)
    JSON list of all pins, by creation date
    private pins in list if user has right or is admin (auth)
    create a new pin (auth)

    "/api/pin/user/<user>/" [GET](any)
    JSON flat list of a user pins pks by creation date
    Private pins if user has right or is admin (auth)

OK  "/api/pin/<pk>/" [GET, PUT, PATCH, DELETE](any, auth to update/delete)
    JSON pin detail
    update or delete pin if admin or owner

OK  "/api/pin/tags/" [GET](any)
    JSON flat list of all tags' slug



## Boards

OK  "/api/board/menu/" [GET](admin)
    JSON list of board api endpoints

OK  "/api/board/" [GET, POST] (admin, auth)
    JSON list of all boards [GET](admin)
    create a new board [POST](auth)

OK  "/api/board/public/user/<user>/" [GET](any)
    JSON list of user public boards abstracts

OK  "/api/board/private/user/<user>/" [GET](authenticated with rights)
    JSON list of user private's boards abstracts

OK  "/api/board/user/<user>/board/<board>/" [GET](any)
    JSON board detail (private board if user has right)
    get a board 

OK  "/api/board/edit/user/<user>/board/<board>/" [GET, PUT, PATCH, DELETE](owner or admin)
    JSON board detail for edition
    update or delete a board if admin or owner
